Command:

Skeleton [fileinput] [filefeatures] [scale]

*************************************************************************************
* Test Skeleton3D C++ application
*************************************************************************************

The Skeleton executable locates at the /usr/local/bin/skeleton3D folder

Unzip rsa-pipeline-skeleton3d-test-data.tar.gz to some folder [my_folder] on your computer.

You will get the following:

[my_folder]/rsa-pipeline-skeleton3d-test-data/                                              
                                             benchmark/ 
                                                       bm-features.tsv 
                                             OsRILp00001d12_rootwork.out


[fileinput] = [my_folder]/rsa-pipeline-skeleton3d-test-data/OsRILp00001d12_rootwork.out
[filefeatures] = [my_folder]/rsa-pipeline-skeleton3d-test-data/features.tsv
[scale] = 2.25

Type in the terminal window:

/usr/local/bin/skeleton3D/Skeleton [my_folder]/rsa-pipeline-skeleton3d-test-data/OsRILp00001d12_rootwork.out [my_folder]/rsa-pipeline-skeleton3d-test-data/features.tsv 2.25

On my computer, I chose for test:

[my_folder] = ~/tmp

So, 

/usr/local/bin/skeleton3D/Skeleton  ~/tmp/rsa-pipeline-skeleton3d-test-data/OsRILp00001d12_rootwork.out  ~/tmp/rsa-pipeline-skeleton3d-test-data/features.tsv 2.25
