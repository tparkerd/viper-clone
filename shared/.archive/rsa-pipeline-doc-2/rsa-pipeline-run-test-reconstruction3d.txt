Command:

Reconstruction [option] [Width] [Height] [NumberNodesOnOctree] [SilhouettePrefix] [NumImgUsed] [ParaFilePathName] [Resolution] [DistortionRadius] [NumComponents] [OutputVoxels] [ExtraInfo] [RotationDigits] [RefImg] [RefRatio]

*************************************************************************************
* Test Reconstruction3D C++ application
*************************************************************************************

The Reconstruction executable locates at the /usr/local/bin/reconstruction3D folder

Unzip rsa-pipeline-reconstruction3d-test-data.tar.gz to some folder [my_folder] on your computer.

You will get the [my_folder]/silhouette3 with images and para.txt file.
(The para.txt file is, in fact, a rotations matrix, used in the reconstruction3D application, for rotations angles 0,1x9,2x9,3x9, ... , where 360/40=9 ):

option = 1
Width = 453
Height = 472
NumberNodesOnOctree = 10000000
SilhouettePrefix = [my_folder]/rsa-pipeline-reconstruction3d-test-data/silhouette3/OsRILp00001d12_
NumImgUsed = 20
ParaFilePathName =  [my_folder]/rsa-pipeline-reconstruction3d-test-data/para.txt
Resolution = 4
DistortionRadius = -1
NumComponents = 1
ExtraInfo = "Extra Information" 
OutputVoxels = [my_folder]/rsa-pipeline-reconstruction3d-test-data/OsRILp00001d12_rootwork.out
RotationDigits = 2
RefImg = 1
RefRatio = 1

Type in the terminal window:

/usr/local/bin/reconstruction3D/Reconstruction  1 453 472 10000000 [my_folder]/rsa-pipeline-reconstruction3d-test-data/silhouette3/OsRILp00001d12_ 20 [my_folder]/rsa-pipeline-reconstruction3d-test-data/para.txt 4 -1 1  [my_folder]/rsa-pipeline-reconstruction3d-test-data/OsRILp00001d12_rootwork.out  "Extra Information"   2 1 1

On my computer, I chose for test:

[my_folder] = ~/tmp

So,
